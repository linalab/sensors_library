# Circuit:
- GND -> GND
- VCC -> 3.3V
- OUT -> AIN0
- // Gain Adjustments:
- GAIN -> VCC ( 50dB )
- GAIN -> GND ( 40dB )
- GAIN -> Disconnected (60 dB )
