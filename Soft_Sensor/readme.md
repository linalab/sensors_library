# Soft Pressure Sensor

## Materials needed:

- Conductive Copper Tape ( or any conductive material )
- Velostat
- Multimeter
- Resistors

## Citcuit

- Sensor Pad 1 -> 5V
- Sensor Pad 2 -> Analogic Pin -> Resistor -> GND
